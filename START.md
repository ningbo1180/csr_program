## ⚡ 60 秒快速启动

### 步骤 1: 启动后端

打开**命令行**或**PowerShell**，运行：

```bash
cd backend
python main.py
```

等待看到输出：
```
INFO:     Uvicorn running on http://0.0.0.0:8000
INFO:     CSR GenAI Backend starting up...
INFO:     Database initialized successfully
```

### 步骤 2: 启动前端

打开**新的命令行窗口**，运行：

```bash
cd frontend
npm run dev
```

等待看到输出：
```
 ▲ Next.js 14.x.x
 - Local:        http://localhost:3000
```

### 步骤 3: 打开浏览器

在浏览器中访问：
```
http://localhost:3000
```

**✅ 应用已准备好使用！**

## 🎯 首次使用工作流

### 1️⃣ 创建你的第一个项目

在主页上：
1. 点击 **"开始新项目"** 或 **"打开已有项目"**
2. 输入项目名称，例如："我的第一个 CSR"
3. （可选）添加描述
4. 点击 **"创建项目"**

**成功！** 你已创建了第一个项目。

### 2️⃣ 上传文档

在项目页面的 **"文档上传"** 标签：

1. 拖拽或点击 **"选择文件"** 上传文档
2. 支持格式: PDF, DOCX, XLSX
3. 等待上传完成（看到 "上传完成" 消息）
4. 查看 **"已上传的文档"** 列表

**成功！** 文档已自动解析。

### 3️⃣ 创建报告结构

在项目页面的 **"结构树"** 标签：

1. 输入章节号，例如: `10.2.6`
2. 输入章节标题，例如: `数据分析结果`
3. 点击 **"添加章节"**
4. 查看 **"报告目录"** 中新增的章节

**成功！** 你已创建了项目结构。

### 4️⃣ 查看操作日志

在项目页面的 **"操作日志"** 标签：

- 查看所有已执行的操作
- 每个操作都有时间戳和详细信息
- 点击 **"刷新"** 获取最新日志

**成功！** 你已完成完整工作流！

## 🆘 常见问题

### Q: 应用无法启动？

**A:** 检查以下几点：

1. **Python 版本** (需要 3.9+)
   ```bash
   python --version
   ```

2. **Node.js 版本** (需要 18+)
   ```bash
   node --version
   ```

3. **重新安装依赖**
   ```bash
   # 后端
   cd backend
   pip install -r requirements.txt
   
   # 前端
   cd frontend
   npm install
   ```

### Q: 创建项目时出错？

**A:** 
1. 确保后端正在运行 (http://localhost:8000 应该可访问)
2. 检查浏览器的开发者工具 (F12 → Console)
3. 查看后端日志输出

### Q: 上传文件失败？

**A:**
1. 确保文件大小 < 50MB
2. 文件格式必须是 PDF, DOCX 或 XLSX
3. 确保 `backend/uploads/` 目录存在（会自动创建）

### Q: 无法看到项目列表？

**A:**
1. 刷新浏览器页面 (Ctrl+R)
2. 打开浏览器开发者工具检查错误
3. 确保后端正常运行

## 🔧 有用的命令

### 重置数据库

如果需要清空所有数据：

```bash
# 1. 停止后端 (Ctrl+C)

# 2. 删除数据库文件
rm backend/csr.db

# 3. 重启后端
python main.py
# 数据库会自动重建
```

### 查看 API 文档

启动后端后，访问：
```
http://localhost:8000/docs
```

在 Swagger UI 中可以测试所有 API 端点。

### 运行自动化测试

**Windows:**
```bash
test_api.bat
```

**Linux/macOS:**
```bash
bash test_api.sh
```
